Besides TuxGuitar’s own file format (with *.tg extension), TuxGuitar can work with various tablature and music file formats.

Tablatures

Guitar Pro 4 (.gp4) - load and save

Guitar Pro 5 (.gp5) - load and save

PowerTab (.ptb) - load

TableEdit (.tef) - load

LilyPond (.ly) - export

MusicXML (.xml) - export
